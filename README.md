# Cookie Manager Plugin for Grav

Advanced customizable cookie consent manager plugin for Grav CMS.

## Features

- Accept / Necessary / Reject buttons
- Google Analytics integration
- Consent-based script loading
- Dynamic layouts
- Dynamic positioning
- Configurable cookie durations
- Responsive design

## Installation

Copy plugin into:

/user/plugins/cookie-manager

OR use GPM after approval.
